# collatz-ratios

Finds integers which take a long time to reach one when verifying the [Collatz conjecture](https://en.wikipedia.org/wiki/Collatz_conjecture) for those integers ([further explanation](https://youtu.be/PrWXtgHCxNk)).

## Smallest ratios found
3, 7, 9, 27, 230631, 626331, 837799, 1723519, 3732423, 6649279, 8400511, 63728127
Update the [original repo](lizard-heart/collatz-ratios) if you find better ones.
